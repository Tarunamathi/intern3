Place PDF files related to the Livelihood section in this folder.

Recommended filename format: `livelihood-<short-name>.pdf` (e.g. `livelihood-case-study-2025.pdf`).

Access example from the app: `/docs/livelihood/livelihood-case-study-2025.pdf`

How to reference in Next.js/React:

- Static link: `<a href="/docs/livelihood/livelihood-case-study-2025.pdf" target="_blank" rel="noopener">Download PDF</a>`
- Programmatic navigation: `router.push('/docs/livelihood/livelihood-case-study-2025.pdf')`

If you want an index page that lists all PDFs automatically, I can add a small component to read a manifest or maintain a JSON list here.
