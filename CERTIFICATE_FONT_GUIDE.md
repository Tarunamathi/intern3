## Troubleshooting

### Certificate doesn't fit on one page
**Solution**: Use "Small" font size
- Reduces all text proportionally
- Still maintains readability
- Guaranteed to fit A4 page

### Text looks too small
**Solution**: Use "Large" font size
- Increases all elements
- Still fits on one page
- Better for presentations

### Font doesn't look right
**Solution**: Try a different font style
- Elegant: Classic serif look
- Modern: Clean sans-serif
- Classic: Traditional serif

### Print preview shows extra blank pages
**Solution**: Check print settings
- Select "Print Selection" if available
- Ensure "Fit to Page" is enabled
- Check page size is set to A4

## Tips & Tricks

1. **Preview Before Printing**
   - Always use "Print Preview" first
   - Check alignment and spacing
   - Adjust if needed

2. **Use Elegant for Formal Settings**
   - Academic institutions
   - Professional certifications
   - Government programs

3. **Use Modern for Contemporary Programs**
   - Tech bootcamps
   - Startup training
   - Creative programs

4. **High Contrast Templates**
   - Premium Gold: Best visibility
   - Ocean Blue: Professional look
   - Vibrant Coral: Energetic vibe

5. **Save Templates**
   - Use same template for consistency
   - Users recognize your branding
   - Professional appearance maintained
