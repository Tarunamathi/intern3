**Q: Why is my certificate not fitting on one page?**
A: Use "Small" font size. It's designed to fit any certificate on A4.

**Q: Can I change fonts after issuing?**
A: No, but you can reissue with different settings and delete the old one.

**Q: Which font size should I use?**
A: Default to "Medium". Change only if needed.

**Q: Which font style should I use?**
A: Default to "Elegant". It looks professional for all situations.

**Q: Does it really fit on one page?**

**Q: Can I use multiple fonts in one program?**
